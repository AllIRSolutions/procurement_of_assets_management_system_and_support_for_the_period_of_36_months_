# municipal-asset-management-system

> Built by **AllIR Solutions** for tender: VDM/FIN/01/03/2026/01

## Overview

Comprehensive asset management system for South African municipalities to track, manage, and report on municipal assets with integrated support functionality and AI-powered assistance

## Features

- Asset lifecycle management (acquisition to disposal)
- Barcode/QR code asset tagging and mobile scanning
- Preventive and corrective maintenance scheduling
- Financial tracking with depreciation calculations
- MFMA and GRAP compliance reporting
- Real-time asset location tracking with GPS integration
- Document management with version control
- Role-based access control for different municipal departments
- Mobile application for field asset management
- Integration with municipal financial systems
- Automated maintenance alerts and notifications
- Asset performance analytics and KPI dashboards
- Bulk asset import/export functionality
- Multi-language support (English, Afrikaans, Local languages)
- Offline capability for field operations
- Asset disposal and transfer workflows
- Insurance tracking and claims management
- Warranty management and tracking

## Tech Stack

- **Backend:** Node.js/Express with TypeScript
- **Frontend:** React + TailwindCSS + Material-UI
- **Database:** PostgreSQL with Redis caching
- **AI:** Claude API integration for asset insights and maintenance recommendations

## API Endpoints

- `GET /api/assets` — Retrieve paginated list of assets with filtering and search
- `POST /api/assets` — Register new municipal asset
- `PUT /api/assets/:id` — Update asset information and status
- `DELETE /api/assets/:id` — Decommission or dispose of asset
- `GET /api/maintenance/schedule` — Get maintenance schedules and overdue items
- `POST /api/maintenance/work-order` — Create maintenance work order
- `GET /api/reports/compliance/:type` — Generate compliance reports (MFMA, GRAP, etc.)
- `GET /api/reports/financial-summary` — Generate financial asset reports and depreciation schedules
- `POST /api/documents/upload` — Upload asset-related documents and attachments
- `GET /api/analytics/dashboard` — Get asset analytics and KPI data for dashboard
- `POST /api/auth/login` — User authentication and JWT token generation
- `GET /api/notifications` — Get user notifications and system alerts
